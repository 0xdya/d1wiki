TimeAr

مكتبة js بالشراكة مع الاسطورة @mahmoudplay لتحويل التواريخ إلى صيغة "منذ..." بالعربية، مع مراعاة قواعد التصريف الصحيحة للأعداد.

المشكلة

أغلب مكتبات الـ "time ago" الموجودة إما تدعم العربية بشكل سطحي، أو تترجم الأرقام ترجمة حرفية دون مراعاة قواعد اللغة. في العربية، الأعداد تُصرَّف مع المعدود: فرق شاسع بين "ثانية واحدة"، "ثانيتان"، "ثلاث ثوانٍ"، و"عشر ثوانٍ". time-ar تحل هذه المشكلة بدقة، وبدون أي تبعيات خارجية.

التثبيت

طريقتان للتثبيت حسب نوع مشروعك:

عبر CDN — للصفحات البسيطة أو المشاريع التي لا تستخدم bundler:

html 
<script src="https://cdn.jsdelivr.net/gh/0xdya/timeAr@latest/cdn/timeAr.js"></script>

عبر npm — للمشاريع التي تستخدم Node.js أو أدوات مثل Vite وWebpack:

bash 
npm i time-ar

الاستخدام

المكتبة تعمل مباشرة على HTML دون الحاجة لكتابة أي كود JavaScript إضافي. كل ما عليك فعله هو إضافة الكلاس time_ago لأي عنصر، ثم تمرير التاريخ بإحدى طريقتين:

الطريقة الأولى: عبر الخاصية data-date — العنصر يبقى فارغًا والمكتبة تملؤه:

html 
<span class="time_ago" data-date="2025-01-15T10:00:00"></span>

الطريقة الثانية: التاريخ مباشرة في نص العنصر — مفيد إذا كانت البيانات تأتي من server-side rendering:

html 
<span class="time_ago">2025-01-15T10:00:00</span>

في الحالتين ستكون النتيجة نصًا مثل: منذ ٣ أشهر، منذ يومين، منذ ٤٥ ثانية.

التاريخ يجب أن يكون بصيغة ISO 8601 — وهي الصيغة الافتراضية التي تُرجعها معظم قواعد البيانات وخدمات الـ API.

لماذا timeAr وليس بديلاً آخر؟

  • تصريف صحيح للأعداد: المكتبة تفرّق بين المفرد والمثنى والجمع بشكل سليم، وهو ما تفتقده معظم البدائل.
  • تعمل بدون كود: مناسبة جداً للمشاريع static ولمن يبني صفحات سريعة أو يدمج المكتبة في CMS.
  • مفتوحة المصدر: يمكنك قراءة الكود وتعديله وفق احتياجاتك.

المحتوى العربي التقني نادر، وكل مشروع مفتوح المصدر بالعربية يستحق الدعم.
