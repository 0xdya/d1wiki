FastPush

أداة Bash مفتوحة المصدر لربط ورفع ملفات مشروعك إلى قيتهاب بسهولة

التثبيت

1
تحميل الملف:
نسخcurl -o fastpush.sh https://raw.githubusercontent.com/0xdya/fastpush/main/fastpush.sh
2
إعطاء الصلاحيات:
نسخchmod +x fastpush.sh

الاستعمال

الأوامر الأساسية:

نسخ./fastpush.sh -m "رسالة الكوميت"
نسخ./fastpush.sh --init
نسخ./fastpush.sh --help

- -m يحدد رسالة الكوميت مباشرة.
- --init يهيئ مشروع جديد ويربطه بمستودع GitHub.
- --help يعرض المساعدة.

تنويه: الأداة تضيف جميع الملفات تلقائيًا. استعمل ملف .gitignore لتجنب رفع ملفات غير ضرورية.
